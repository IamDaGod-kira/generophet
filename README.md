# Generophet

Generophet is a modern, interactive genetic inheritance simulator and educational tool. It allows users to visualize Mendelian genetics through Punnett squares, explore probability ratios, and learn complex genetic concepts with an integrated AI tutor.

## Features

-   **Interactive Punnett Square Generator**: Visualizes crosses for up to 5 genes.
-   **Step-by-Step Walker**: Breaks down the calculation process for better understanding.
-   **AI Tutor**: Integrated "Learn" section powered by Google Gemini AI to answer genetics questions.
-   **Customizable Traits**: Define your own genes, alleles, and phenotypes.
-   **Ratio Analysis**: Automatically calculates genotypic and phenotypic ratios.
-   **Vibrant UI**: Designed with a modern, glassmorphic aesthetic using Tailwind CSS.

## Getting Started

### Prerequisites

-   Node.js (v18 or higher)
-   npm or yarn

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/IamDaGod-kira/geneprophet.git
    cd geneprophet
    ```

2.  Install dependencies:
    ```bash
    npm install
    ```

3.  Configure API Key:
    -   Create a `.env` file in the root directory.
    -   Add your Google Gemini API key:
        ```bash
        VITE_GOOGLE_AI_KEY=your_api_key_here
        ```

4.  Run the development server:
    ```bash
    npm run dev
    ```
